Ensuite, pour créer une variable, on fait:

var name: type = value
          ^^^^
          Les types est entre ':' et '='
Dans les fonctions, c'est plus:

function name(arg1: type, arg2: type)(functiontype)
                    ^^^^              ^^^^^^^^^^^^
                    Type de l'arg     Type de la fonction
